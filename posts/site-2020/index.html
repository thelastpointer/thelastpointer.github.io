Keeping the site up-to-date

This site, despite not getting any traffic, is a project I very much like, and periodically I feel the need to polish it.

First of all, I changed my gallery to Fotorama. It is very simple, relatively lightweight, and supports everything I need – mobile view, fullscreen and thumbnails. That also means I could get rid of the old gallery, which mysteriously stopped working. I think it was the Hugo version upgrade?

Whatever. I also added a link to Herogrinder’s Steam page on my CV to keep things up-to-date. I’ll probably add some more images too.

By far the biggest change is the migration to GoatCounter – this is a web analytics platform that is lightweight and does not require fucking cookies. So I can ditch Google Analytics and the extremely useless cookie consent popup! Yay! But seriously, that project is amazing, and I can 100% stand behind Marting’s rationale for creating it.

Sadly, this year has been busy in a not-really-spectacular way, so I have little to post. There are some interesting things in the pipeline – my first electronics project, some coding, some game design, so while I’ll never get to last years post count, I hope I can add some useful content soon.
